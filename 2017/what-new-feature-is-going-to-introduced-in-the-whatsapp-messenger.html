What new feature is going to introduced in the Whatsapp messenger?

(A) Animated GIF

(B) Status updates

(C) ** Changing number

(D) Video call

EXPLANATIONS BELOW

Concept note-1: -Meta has implemented a lot of security measures to make WhatsApp a safe and secure messaging platform. One such feature is the ability to hide online status, this enables users to stay online on WhatsApp and control who can see their current status.

Concept note-2: -Alongside Communities, WhatsApp introduced new group messaging features, including the ability to create in-chat polls, the option to add up to 1, 024 members, up from 256. It also introduced admin delete, giving admins the ability to delete problematic messages. This year, WhatsApp launched add-ons to this feature.

Concept note-3: -In 2023, WhatsApp is expected to launch the companion mode, with the help of which users will get the option to link and sync their WhatsApp data and use the app with the same profile on several phones. It will also let the users use the same WhatsApp profile simultaneously across Android and iOS devices.
