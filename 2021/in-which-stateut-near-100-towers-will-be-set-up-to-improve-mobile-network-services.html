In which state/UT near 100 towers will be set up to improve Mobile Network Services?

(A) ** Ladakh

(B) Delhi

(C) Sikkim

(D) Bihar

EXPLANATIONS BELOW

Concept note-1: -Nearly 100 towers to be installed in Ladakh to improve mobile network services.

Concept note-2: -The first point to remember is tower installation is NOT illegal in residential areas. However, if there are lack of site options, a tower can be installed in a residential area. Also it is important to note that if a tower must be installed in a residential area, it should only be on rooftops of buildings.

Concept note-3: -The antenna rule As per DoT guidelines, if a mobile tower has 1 antenna, it should be installed at minimum distance of 20 metres from a house, in case of 2 antennas minimum distance should be 35 metres, for 4 antennas it should be 45 metres and for 6 antennas the minimum distance should be 55 metres.

Concept note-4: -The government has already approved 980 4G mobile towers for the state, and the project is being implemented by Bharti Hexacom Ltd.
