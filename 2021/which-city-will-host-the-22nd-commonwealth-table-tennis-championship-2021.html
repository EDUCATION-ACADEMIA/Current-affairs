Which city will host the 22nd Commonwealth Table Tennis Championship 2021?

(A) New Delhi

(B) ** Panchkula

(C) Guwahati

(D) Mumbai

EXPLANATIONS BELOW

Concept note-1: -Which city will host the 22nd Commonwealth Table Tennis Championship 2021? Explanation: The Panchkula will host the Commonwealth table tennis championship later this year.

Concept note-2: -A WTT event is set to be held in Goa next month. Indian won seven medals while England won five in the table tennis tournament held at the Birmingham 2022 Commonwealth Games.
