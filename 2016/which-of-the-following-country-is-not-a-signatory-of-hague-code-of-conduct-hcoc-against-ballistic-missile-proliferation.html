Which of the following country is not a signatory of Hague Code of Conduct (HCoC) against Ballistic Missile Proliferation

(A) Argentina

(B) Russia

(C) ** Pakistan

(D) Chile

EXPLANATIONS BELOW

Concept note-1: -China is currently the main ballistic missile possessor and spacefaring nation which remains outside the Hague Code of Conduct against Ballistic Missile Proliferation (HCoC).

Concept note-2: -The HCOC is aimed at bolstering efforts to curb ballistic missile proliferation worldwide and to further delegitimize such proliferation. The HCOC consists of a set of general principles, modest commitments, and limited confidence-building measures.

Concept note-3: -India has recently joined the Hague Code of Conduct [HCOC], a global initiative to prevent ballistic missile proliferation.

Concept note-4: -The International Code of Conduct against Ballistic Missile Proliferation, also known as the Hague Code of Conduct (HCOC), was established on 25 November 2002 as an arrangement to prevent the proliferation of ballistic missiles.

Concept note-5: -What is Hague Code of Conduct? It seeks to prevent the proliferation of ballistic missiles that are capable of delivering weapons of mass destruction (WMD). It does not ban ballistic missiles, but it calls for restraining their production, testing, and export.
