Elite x3 smartphone runs on which version of Windows

(A) Windows 8

(B) Windows 8.5

(C) Windows 9

(D) ** Windows 10

EXPLANATIONS BELOW

Concept note-1: -Elite x3 Mobile Device – This is the ‘smartphone’ running Windows 10 Mobile, powered by a Qualcomm Snapdragon 820 CPU with 64GB of internal memory.

Concept note-2: -Windows 10 Mobile was designed for use on smartphones and phablets running on 32-bit ARM processor architectures.

Concept note-3: -Windows 10 Mobile is an edition of the Windows 10 operating system developed by Microsoft for use on mobile devices including smartphones and small tablets under 8 inches in screen size.
